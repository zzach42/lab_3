# Создайте сущность Имя, которая описывается тремя параметрами: Фамилия, Личное имя, Отчество.
Имя может быть приведено к строковому виду, включающему традиционное представление всех трёх параметров:
Фамилия Имя Отчество (например, «Иванов Иван Иванович»).

Необходимо предусмотреть возможность того, что какой-либо из параметров может быть не задан.
В этом случае он не учитывается при приведении имени к строковому виду.

Необходимо создать следующие имена:

Клеопатра

Пушкин Александр Сергеевич

Маяковский Владимир

При выводе на экран не заданные параметры не участвуют в формировании строки.

Класс Name

Поля (private):

first_name_ — личное имя

last_name_ — фамилия

patronymic_ — отчество

Методы (public):

Конструкторы для задания имени, фамилии и отчества

toString() — формирует строковое представление имени

print() — выводит имя на экран

Алгоритм работы кода

Создаются объекты Name с разным количеством заданных параметров.

Метод toString():

Проверяет каждое поле.

Если поле не пустое — добавляет его в результирующую строку.

Поля соединяются пробелами.

Метод print() выводит итоговую строку в консоль.

Пример выполнения
Клеопатра
Пушкин Александр Сергеевич
Маяковский Владимир

Задача 3. Сотрудники и отделы (старая версия)
Условие

# Создайте сущности Сотрудник и Отдел.

Каждый сотрудник:

имеет имя,

работает в отделе.

Каждый отдел:

имеет название,

имеет начальника (который тоже является сотрудником).

Текстовое представление сотрудника:

Если сотрудник начальник:
Имя начальник отдела Название

Если обычный сотрудник:
Имя работает в отделе Название, начальник которого Имя

Необходимо:

Создать сотрудников Петрова, Козлова, Сидорова в отделе IT.

Назначить Козлова начальником отдела.

Вывести всех сотрудников.

Класс Employee

Поля (private):

name_ — имя сотрудника

department_ — указатель на отдел

Методы (public):

Конструктор Employee(name)

setDepartment(department)

toString()

print()

Класс Department

Поля (private):

name_ — название отдела

boss_ — указатель на начальника

Методы (public):

Конструктор Department(name)

setBoss(employee)

getName()

getBoss()

Пример выполнения
Петров работает в отделе IT, начальник которого Козлов
Козлов начальник отдела IT
Сидоров работает в отделе IT, начальник которого Козлов
